Course materials¶

All of my lecture notes are publicly accessible, either through SpeakerDeck (for slides) or Gist (for IPython Notebooks, python scripts, data sets, etc). See the Lecture notes column in the course schedule to find a link to the materials for that lecture.

There is no text book for this class. All of the reading materials are publicly available on the web, and you can find them linked from the course schedule.

Find the course syllabus here.

The IPython Notebook server used in class is accessible here. You’ll get the password in class. Note that you can only access this system from NAU IP addresses, included ethernet connections on campus, via the NAU wireless network, or via VPN when off-campus. Thanks to ITS Academic Computing for setting this up for us!
